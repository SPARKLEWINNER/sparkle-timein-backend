# MERN Sparkle Time-In

## What's included?
 - Node Express Microservice
 - Mongo -> Mongoose
 - Socket IO
 - React
 - JWT Token
 - Google SDK
 - Firebase SDK

## Setup
 - npm install (root)
 - cd react-src && npm install (react)

## Folder Structure
 - api
   - controller
   - models
   - helpers
   - validators
   - services
   - route.js
 - react-src
 - public (automatically gets the build of react-src for public view)
 - server.js
 - .env
